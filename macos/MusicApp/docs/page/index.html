Equalizer

Your headphones, perfected

A 10-band parametric equalizer with peak and low/high shelf filters runs directly in the audio pipeline. Rather than tweaking sliders by ear, choose from over 100 calibration presets based on oratory1990 measurements, each targeting the Harman frequency response curve for your specific headphone model.

Bands 10 parametric (peak + shelf)

Library

Your music, organized automatically

Point FlaYer at your music folders and it does the rest. Metadata is extracted from every file — title, artist, album, genre, year, track number, cover art, sample rate, bit depth — and stored in a local SQLite database. A file system watcher monitors your folders in real time: add, move, or delete files and your library updates itself.

Search

Full-text search powered by FTS5 with Unicode tokenization. Accent-insensitive, prefix matching, instant results across artists, albums, tracks and genres. Recent searches are saved for quick access.

Browsing

Albums and artists are displayed as cover art grids with multiple sort modes: alphabetical, by year, or grouped by time period (today, this week, this month). Create playlists, mark favorites, and sort tracks by any field.

Interface

Dark, minimal, yours

A pure black canvas with translucent glass elements. No chrome, no clutter. The interface stays out of the way and lets album art and typography do the work. Everything is built natively in SwiftUI — no Electron, no web views.

Under the hood

Native from top to bottom

FlaYer is written entirely in Swift 6 with SwiftUI. Audio runs through AVAudioEngine with CoreAudio device management. The database is SQLite via GRDB with full-text search, migrations, and concurrent access. File watching uses FSEvents for zero-polling folder monitoring.

Language Swift 6.0
UI SwiftUI
Audio AVAudioEngine + CoreAudio
Database SQLite (GRDB 7)
